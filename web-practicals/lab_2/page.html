Реєстрація на курс

Заповніть форму нижче, щоб зареєструватися на наш безкоштовний інтенсив з веб-розробки. Поля з зірочкою — обов'язкові.

Курс триває 6 тижнів. Заняття — щовівторка і четверга. Після курсу ви отримаєте сертифікат і доступ до закритого Telegram-чату випускників.

Особисті дані

Що ви опануєте

Основні блоки програми курсу:

  • HTML і семантика веб-сторінки
  • CSS — оформлення, селектори, адаптивність
  • JavaScript — синтаксис, типи, функції
  • DOM — взаємодія з елементами сторінки
  • HTTP і робота з API

А також додаткові інструменти:

  • Git і GitHub
  • DevTools і дебагінг
  • Базові основи TypeScript
  • Веб-безпека: XSS, CSRF, MitM
  • Деплой простих застосунків
